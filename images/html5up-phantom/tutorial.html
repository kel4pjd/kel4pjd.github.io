A. Menentukan Dataset

Data yang digunakan merupakan data dari Badan Pusat Statistik (BPS) Kabupaten Bantul dengan proses pencarian sebagai berikut:

  1. Membuka laman BPS Kabupaten Bantul

    2. http://html5up.net

  2. Membuka data Sosial dan Kependudukan bagian kesehatan
  3. Memilah 1 dataset dari 5 dataset yang tertampil, yaitu dataset dengan judul “Prevalensi Jenis Penyakit per Kecamatan, 2018–2021”

    4. Dataset yang dipilih berisi data prevalensi penyakit dari 18 Kecamatan di Kabupaten Bantul selama periode 2019–2021. Data ini mencakup informasi mengenai TB Paru, Pneumonia Balita, DBD, dan Diare.
